接口：UserMessage.getMessageList


获取留言及评论
接口说明

获取留言及评论

接口参数

参数名字 类型 是否必须 默认值 其他 说明
token 字符串 必须 最小：1 时间标签
page_size 整数 可选 15 每页显示记录
page_no 整数 可选 1 当前的页

返回结果

返回字段 类型 说明
data.code 整型 操作码，
code = 0
code = 1
data.code_dec 字符串 返回代码描述
data.list[n] array 留言数据：
uid 用户id
username 用户名
nickname 昵称
header 头像
pictures[n] 图片列表，n=0~8,最多9附图
comment[n] 评论列表
m_id 留言id
message 留言内容
time 留言时间
is_del 是否可以删除（仅仅发表人可以删除）
data.pageNo 整数 当前的页
data.page_size 整数 每页显示记录
