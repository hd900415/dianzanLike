接口：User.getuserinfo


获取会员信息
接口说明

获取会员信息

接口参数

参数名字类型是否必须默认值其他说明
token字符串必须 最小：1时间标签

返回结果

返回字段类型说明
data.code整型操作码，
code = 0 您输入的密聊号不正确
code = 1
data.info.userid字符串用户id
data.info.username字符串用户名称
data.info.balance字符串余额
data.info.frozen_balance数字冻结余额
data.info.state字符串账户状态
data.info.phone字符串手机号
data.info.bankcount字符串绑定银行卡数量
data.info.qrcodcount字符串绑定二维码数量
