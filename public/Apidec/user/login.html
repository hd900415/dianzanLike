接口：User.Login


用户登录接口
接口说明

用户使用该接口登录平台

接口参数

参数名字类型是否必须默认值其他说明
username数字必须 最小：11用户名，这里用户名使用手机号
password字符串必须 最小：6 最大：16用户密码

返回结果

返回字段类型说明
data.code整型操作码，
code = 0 用户名、密码或者平台sign为空
code = 1 登录成功
code = 2 该账号已锁定
code = 3 注册成功
code = 4 注册失败
data.code_dec字符串返回代码描述
data.info.userid数字用户id
data.info.username字符串用户名
data.info.balance数字用户余额
data.info.frozen_balance数字冻结余额
data.info.state数字用户账户状态。2冻结
data.info.token字符串时间标签
data.info.bankcount字符串绑定银行卡的数量
data.info.qrcodcount数字绑定二维码的数量
