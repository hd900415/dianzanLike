接口：User.getuserdata


获取用户数据
接口说明

获取用户数据

接口参数

参数名字 类型 是否必须 默认值 其他 说明
token 字符串 必须 最小：1 时间标签
uid 整数 必须 最小：6 最大：16 用户ID

返回结果

返回字段 类型 说明
data.code 整型 操作码，
code = 0
code = 1
data.code_dec 字符串 返回代码描述
data.mldata[n] array 密聊数据：
title 标题(私聊用户名/用户昵称)（群聊群名称)
content 聊天内容最新一条数据的
types 聊天类型 私聊=1；群聊=2 ：用户查看聊天记录用
time 聊天聊天时间 is_see_num 未读数
data.is_see array 通知记录数：
ml_num 密聊未读数
yl_num 友聊未读数
ql_num 群聊未读数
