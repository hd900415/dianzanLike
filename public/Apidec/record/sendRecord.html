接口：Record.sendRecord


用户发送聊天记录
接口说明

用户发送聊天记录

接口参数

参数名字 类型 是否必须 默认值 其他 说明
token 字符串 必须 最小：1 时间标签
uid 整数 必须 最小：6 最大：16 用户ID
id 整数 必须 最小：6 最大：16 友聊就是好友表Id/群聊就是群表Id
chat_type 整数 必须 最小：6 最大：16 聊天类型 / 1是友聊/2群聊/3好友添加回复（不显示在密聊，只显示在好友邀请回复）/ 4群公告 @所以人 /5修改群名称 6/修改群公告
content 整数 必须 最小：6 最大：16 聊天内容
content_type 整数 必须 最小：6 最大：16 内容类型1文本/2图片/3名片4/红包

返回结果

返回字段 类型 说明
data.code 整型 操作码，
code = 0
code = 1
data.code_dec 字符串 返回代码描述
