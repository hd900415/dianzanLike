接口：Record.getRecordList


获取聊天记录
接口说明

获取聊天记录

接口参数

参数名字 类型 是否必须 默认值 其他 说明
token 字符串 必须 最小：1 时间标签
uid 整数 必须 最小：6 最大：16 用户ID
id 整数 必须 最小：6 最大：16 友聊就是好友表Id/群聊就是群表Id
chat_type 整数 必须 最小：6 最大：16 聊天类型 / 1是友聊/2群聊/3好友添加回复（不显示在密聊，只显示在好友邀请回复）/ 4群公告 @所以人 /5修改群名称 /6修改群公告
page_no 时间 可选 1 最小：1 页数
page_size 时间 可选 10 最小：1 每页显示条数

返回结果

返回字段 类型 说明
data.code 整型 操作码，
code = 0
code = 1
data.code_dec 字符串 返回代码描述
data.list[n] array tit 标题(私聊用户名/用户昵称)
title 昵称
num 群成员数
header 头像
content 聊天内容
is_send 是否是自己发的聊天1是/0
chat_type 聊天类型 / 1是友聊/2群聊/3好友添加回复（不显示在密聊，只显示在好友邀请回复）/ 4群公告 @所以人 /5修改群名称 /6修改群公告
time 聊天聊天时间
data_total_nums 整型 总数据条数
data_total_page 整型 总页数
data_current_page 整型 当前页
