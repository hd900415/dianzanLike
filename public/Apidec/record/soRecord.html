接口说明

搜

接口参数

参数名字 类型 是否必须 默认值 其他 说明
token 字符串 必须 最小：1 时间标签
uid 整数 必须 最小：6 最大：16 用户ID
content 整数 可选 最小：6 最大：16 搜的内容

返回结果

返回字段 类型 说明
data.code 整型 操作码，
code = 0
code = 1
data.code_dec 字符串 返回代码描述
dataArray.friends[n] array 密友数据：
id 好友表id
nickname 成员昵称
header 头像
mid 米聊号
dataArray.groups[n] array 群数据：
id 群表id
group_name 群名称
header 头像
gid 米群号
dataArray.record[n] array 群数据：
title 标题(私聊用户名/用户昵称)（群聊群标题）
header 头像
recordId ID
record_types 聊天类型 私聊=1；群聊=2
countNo 相关记录数
