接口：Recharge.getRechargeOrderList


获取充值订单列表
接口说明

获取充值订单列表

接口参数

参数名字类型是否必须默认值其他说明
token字符串必须登录标志
pageSize数字可选默认值=10每次获取N条记录
pageNo数字可选默认值=1当前页号

返回结果

返回字段类型说明
data.code整型操作码，
code = 0
code = 1
data.code_dec字符串返回代码描述
data.list.key.id数字订单表id
data.list.key.money数字充值金额
data.list.key.time数字充值时间
data.list.key.state数字订单状态
