接口：Recharge.getRechargeOrderInfo


获取充值订单信息
接口说明

获取充值订单信息

接口参数

参数名字类型是否必须默认值其他说明
token字符串必须登录标志
rechargeOrder_id数字必须充值订单表id

返回结果

返回字段类型说明
data.code整型操作码，
code = 0
code = 1
data.code_dec字符串返回代码描述
data.info.id数字订单表id
data.info.order_number字符串订单号
data.info.money数字充值金额
data.info.u_name字符串收款人姓名
data.info.account_receivable数字收款人账号
data.info.bank_name字符串收款人银行名称
data.info.state数字订单状态
data.info.fee数字手续费
