接口：Account.ChangePassword


修改用户登录密码接口
接口说明

会员修改登录用户密码

接口参数

参数名字类型是否必须默认值其他说明
sign字符串必须签名
token字符串必须时间标签
password_o字符串必须最小：6 最大：16资金密码（旧）
password_n字符串必须最小：6 最大：16资金密码（新）
re_password_n字符串必须最小：6 最大：16确认资金密码（新）

返回结果

返回字段类型说明
code 整型 操作码，
code = 0 表示密码修改失败
code = 1 表示密码修改成功
code = 2 密码格式错误（密码：6-16位，字母、数字组成。必须字母开头,字母必须小写）
code = 3 两次密码不一致
code = 4 旧密码不正确
修改密码成功系统清除用过已经存在的token 需要重新登录
