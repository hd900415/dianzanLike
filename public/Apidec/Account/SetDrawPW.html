接口：Account.setDrawPW


设置支付密码
接口说明

设置支付密码

接口参数

参数名字类型是否必须默认值其他说明
token字符串必须登录标志
drawword_n字符串必须支付密码
re_drawword_n数字必须支付密码

返回结果

返回字段类型说明
data.code整型操作码，
code = 0 设置失败
code = 1 设置成功
code = 2 密码格式错误(6-16位，字母、数字组成。必须字母开头,字母必须小写)
code = 3 两次密码不一致
code = 4 资金密码已经设置
code = 5 资金密码不能与用户密码一致
code = 7 资金密码不能与用户密码一致
data.code_dec字符串返回代码描述
