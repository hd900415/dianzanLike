接口：Account.GetBankCardInfo


获取银行卡信息
接口说明

获取银行卡信息

接口参数

参数名字类型是否必须默认值其他说明
token字符串必须登录标志
card_no数字必须银行卡卡号

返回结果

返回字段类型说明
data.code整型操作码，
code = 0
code = 1
data.code_dec字符串返回代码描述
data.key.id数字记录的id
data.key.name字符串持卡人名称
data.key.card_no数字银行卡卡号
data.key.bank_name字符串银行名称
data.key.bank_branch_name字符串支行名称
