接口：Groups.setGroupsInfo


设置群消息
接口说明

设置群消息

接口参数

参数名字 类型 是否必须 默认值 其他 说明
token 字符串 必须 最小：1 时间标签
gid 字符串 必须 最小：1 群id
uid 整数 必须 用户ID
password 字符串 必须 最小：6 群密码
group_name 字符串 可选 群名称(群主,管理 修改)
is_f_d 整数 可选 管理加群需要验证(群主,管理 显示/修改)
is_h_d 整数 可选 管理允许发送名片(群主,管理 显示/修改)
is_q_d 整数 可选 管理允许通过群号找到群(群主,管理 显示/修改)
is_g_d 整数 可选 管理允许成员邀请好友加群(群主,管理 显示/修改)
is_s_d 整数 可选 管理允许查看成员信息(群主,管理 显示/修改)
is_u_d 整数 可选 管理是否通过共享连接(群主,管理 显示/修改)
notice 字符串 可选 公告(群主,管理 修改)
introduce 字符串 可选 简介(群主,管理 修改)
nick_name 字符串 可选 所在群的昵称
is_v_d 整数 可选 消息勿扰
is_z_d 整数 可选 消息置顶
user_type 整数 可选 成员类型：群主1，成员2，申请3,群管理4
setGroupAdmin 整数 可选 设置群管理员
remove_groups 整数 可选 移出群成员
clear_groups_record 整数 可选 清除群聊天记录
disbanded_groups 整数 可选 群解散
setGroupPassword 字符串 必须 设置群密码

返回结果

返回字段 类型 说明
data.code 整型 操作码，
code = 0
code = 1
data.code_dec 字符串 返回代码描述
