接口：Groups.getGroupsInfo


获取群消息
接口说明

获取群消息

接口参数

参数名字类型是否必须默认值其他说明
token字符串必须 最小：1时间标签
gid字符串必须 最小：1群id
uid整数必须 最小：6 最大：16用户ID

返回结果

返回字段类型说明
data.code整型操作码，
code = 0
code = 1
data.code_dec字符串返回代码描述
data.info.id字符串群ID
data.info.group_name字符串群名称
data.info.gid字符串米群号
data.info.guid字符串群主ID
data.info.is_f_d字符串管理加群需要验证(群主,管理 显示/修改)
data.info.is_h_d字符串管理允许发送名片(群主,管理显示/修改)
data.info.is_q_d字符串管理允许通过群号找到群(群主,管理 显示/修改)
data.info.is_g_d字符串管理允许成员邀请好友加群(群主,管理 显示/修改)
data.info.is_s_d字符串管理允许查看成员信息(群主,管理 显示/修改)
data.info.is_u_d字符串管理是否通过共享连接(群主,管理 显示/修改)
data.info.qun_url字符串管理群链接(群主,管理 修改);如果is_u_d=1 群成员可看到群的链接 发送群链接好友可以通过群链接加入该群
data.info.code_url字符串二维码链接(群主,管理 修改)
data.info.notice字符串公告(群主,管理 修改)
data.info.introduce字符串简介(群主,管理 修改)
data.info.nickname字符串所在群的昵称
data.info.user_type字符串成员类型：群主1，成员2，申请3,群管理4
data.info.is_v_d字符串消息互扰
data.info.is_z_d字符串消息置顶
data.info.state字符串是否显示，1=显示，2=不显示，3=申请中，4=禁言，5=拒绝申请
