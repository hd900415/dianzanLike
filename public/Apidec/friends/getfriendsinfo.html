接口：Friends.getfriendsinfo


获取密友信息
接口说明

获取密友信息

接口参数

参数名字 类型 是否必须 默认值 其他 说明
token 字符串 必须 最小：1 时间标签
uid 整数 必须 最小：6 最大：16 用户ID
id 字符串 必须 最小：6 最大：16 好友列表id

返回结果

返回字段 类型 说明
data.code 整型 操作码，
code = 0
code = 1
data.code_dec 字符串 返回代码描述
data.info array 密聊数据：
id id
user_id Uid
friend_id 好友ID
nickname 好友昵称
header 头像
mid 密号(搜索后显示)
autograph 签名
sex 性别
area 地区
is_v_d 消息免打扰;单独对这个好友接收的聊天不提示声音
is_z_d 置顶聊天
get_type 通过哪种方式查看好友/如果新密友显示回复框，对接获取回复记录接口
new_type 新密友 通知类型 '已请求', //1 '查看', //2 '已通过', //3 '已添加', //4 '被拒绝', //5 '已拒绝', //6 '已删除', //7 '被删除', //8
state 好友状态
'通过', //1 '管理员删除', //2 '审核', //3 '拒绝', //4 '删除', //5
is_p //1主动添加好友,2被动添加好友
状态值说明 3 审核：is_p=2被发送方：用通过/拒绝
调用setfriendsinfo设置好友信息 通过state=3; 拒绝 state=4; 删除state=5
