接口：Activity.getUserActivityList


获取用户活动记录列表
接口说明

获取用户活动记录列表

接口参数

参数名字类型是否必须默认值其他说明
token字符串必须登录标志
page_size数字可选10每页显示记录数量
page_no数字必须当前的页号

返回结果

返回字段类型说明
data.code整型操作码，
code = 0
code = 1
data.code_dec字符串返回代码描述
data.data_total_nums数字总记录数量
data.data_total_page数字总页数
data.data_current_page数字当前页号
data.info.key.id字符串用户活动记录id
data.info.key.date字符串活动时间
data.info.key.orderNum字符串订单号
data.info.key.betTotal字符串当日投注总额
data.info.key.rebate字符串返点值
data.info.key.price字符串金额
